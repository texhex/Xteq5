

If this folder is located at "%PROGRAMDATA%\TestUtil\" (e.g. C:\ProgramData\TestUtil\), 
any file that was installed by SETUP.EXE will be overwritten **without notice**. 
This also includes all files starting with “_” (underscore). 

Therefore, please do not change any existing file or and 
do not name your custom files beginning with “_” (underscore).

Files that you **add** will not be touched.
